## Design Standards

### Hero Sections
- Gradient backgrounds matching app category
- Clear app name and tagline
- Key features highlighted
- Download CTA button

### Feature Cards
- 4-6 feature cards per app
- Icon, title, and description
- Hover effects
- Grid layout (responsive)

### SEO Optimization
- Meta description tags
- Open Graph tags for social sharing
- Twitter Card tags
- Keyword optimization

### Testimonials
- 3 testimonials per app
- Indian names and locations
- Realistic use cases
- Social proof elements

### Navigation
- Footer links to Privacy, Terms, Support
- Consistent across all pages
- Back to home links on subpages

- **Framework:** Pure HTML/CSS (no dependencies)

- **Browser Support:** All modern browsers
- **File Size:** 2-5KB per page (optimized)
- **Accessibility:** Semantic HTML structure
