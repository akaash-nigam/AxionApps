# Frequently Asked Questions

## General

**Q: What is Retail Space Optimizer?**
A: A visionOS app for retail store planning, layout optimization, and customer analytics using immersive 3D visualization.

**Q: What devices are supported?**
A: Apple Vision Pro with visionOS 2.0 or later.

**Q: Is an internet connection required?**
A: Internet is required for syncing and cloud features, but you can work offline with local data.

**Q: How much does it cost?**
A: See [PRICING.md](PRICING.md) for current pricing tiers. Free trial available.

## Features

**Q: Can I import existing store plans?**
A: Yes, import from CAD files (DWG, DXF) or upload floor plan images coming in v0.2.

**Q: Does it support real-time collaboration?**
A: Multi-user collaboration is planned for v0.3.

**Q: Can I export my designs?**
A: Yes, export to PDF, PNG, or 3D formats (USDZ, OBJ).

**Q: Is there a fixture library?**
A: Yes, includes standard retail fixtures. Custom fixtures coming in v0.2.

## Analytics

**Q: Where does analytics data come from?**
A: Integrate with your POS system, or use simulated data based on industry benchmarks.

**Q: How accurate are customer flow simulations?**
A: Simulations use AI models trained on real retail data, typically 85%+ accuracy.

**Q: Can I see real-time analytics?**
A: Real-time analytics require POS integration. Historical data always available.

## Technical

**Q: How many fixtures can I add?**
A: Recommended limit is 100 fixtures per store for optimal performance.

**Q: Does it work offline?**
A: Yes, but cloud sync and real-time collaboration require internet.

**Q: Is my data secure?**
A: Yes, data is encrypted at rest (AES-256) and in transit (TLS 1.3). See [SECURITY.md](SECURITY.md).

**Q: Can I use it on multiple devices?**
A: Yes, CloudKit sync keeps data consistent across devices.

## Troubleshooting

**Q: App crashes on launch**
A: See [TROUBLESHOOTING.md](TROUBLESHOOTING.md) for solutions.

**Q: Fixtures not appearing in 3D view**
A: Ensure fixture models are downloaded. Check Settings > Storage.

**Q: Performance is slow**
A: Reduce fixture count or lower graphics quality in Settings.
