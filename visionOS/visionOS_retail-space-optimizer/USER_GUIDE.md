# Retail Space Optimizer - User Guide

Complete guide for using Retail Space Optimizer on Apple Vision Pro.

## Getting Started

### First Launch

1. Put on your Apple Vision Pro
2. Find "Retail Space Optimizer" in your apps
3. Tap to launch
4. The app starts with sample store data

### Main Interface

**Main Window** - Browse and manage your stores
- Store list with thumbnails
- Create new stores
- Open existing stores
- View analytics summaries

## Creating a Store

1. Click **"New Store"** button
2. Fill in store details:
   - Name
   - Address and location
   - Dimensions (width, length, height)
3. Click **"Create"**
4. Your new store appears in the list

## Editing Store Layouts

### Opening the Editor

1. Select a store from the list
2. Click **"Edit Layout"**
3. A new window opens with the 2D editor

### Using the 2D Editor

**Left Sidebar** - Fixture library
- Browse available fixtures
- Drag fixtures onto canvas
- Filter by category

**Main Canvas** - Store floor plan
- Drag to pan
- Pinch to zoom
- Tap fixtures to select
- Drag selected fixtures to move

**Right Panel** - Properties
- Adjust fixture position
- Rotate fixtures
- Modify dimensions

### Adding Fixtures

1. Browse fixture library
2. Tap and drag a fixture onto the canvas
3. Position where desired
4. Adjust rotation using rotation handle

### Modifying Fixtures

1. Tap to select a fixture
2. Use handles to rotate
3. Edit properties in right panel
4. Delete with trash icon

## 3D Visualization

### Opening 3D Preview

1. Select a store
2. Click **"3D Preview"**
3. A volumetric window appears

### Interacting in 3D

- **Rotate**: Drag with one hand
- **Scale**: Pinch with two hands
- **Pan**: Drag with two hands

### Volume Controls

- Toggle walls visibility
- Toggle grid
- Adjust lighting
- Change view angle

## Immersive Walkthrough

### Entering Immersive Mode

1. Select a store
2. Click **"Walkthrough"**
3. Confirm to enter immersive space
4. You're now "inside" the store

### Navigation

- **Walk**: Use hand gestures or gaze+pinch
- **Look**: Turn your head naturally
- **Interact**: Pinch to select items

### Viewing Analytics

While in immersive mode:
- Toggle heat map overlay
- View customer paths
- See dwell zones
- Display performance metrics

### Exiting

- Look at exit button
- Pinch to activate
- Returns to windowed mode

## Analytics Dashboard

### Opening Analytics

1. Select a store
2. Click **"Analytics"**
3. Dashboard window opens

### Key Metrics

- Total visitors
- Conversion rate
- Sales per square foot
- Average dwell time

### Heat Maps

Choose metric type:
- Traffic (foot traffic density)
- Dwell (time spent in areas)
- Conversion (where purchases happen)
- Sales (revenue by location)

Adjust time range:
- Last 7 days
- Last 30 days
- Custom range

### Customer Journeys

- View individual customer paths
- Filter by persona type
- Analyze conversion patterns
- Identify bottlenecks

## A/B Testing

### Creating a Test

1. Open a store
2. Click **"A/B Test"**
3. Select two layouts to compare
4. Set test duration
5. Start test

### Viewing Results

- Side-by-side comparison
- Statistical significance
- Performance deltas
- Recommendations

## Settings

### User Preferences

- Default view mode (2D/3D/Immersive)
- Units (metric/imperial)
- Grid settings
- Auto-save frequency

### Account

- Profile information
- Organization
- Subscription tier
- Usage statistics

## Tips & Best Practices

### Layout Design

- Leave 40% open space for circulation
- Position high-value items at eye level
- Create focal points at store entrance
- Avoid dead-end aisles

### Using Analytics

- Check heat maps weekly
- Test layout changes with A/B tests
- Focus on conversion, not just traffic
- Monitor seasonal trends

### Performance

- Keep fixture count under 100 for smooth performance
- Use simplified models for large stores
- Clear cache periodically
- Update to latest version
