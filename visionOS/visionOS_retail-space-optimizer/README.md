## üì± Quick Start

### Prerequisites

- macOS Sonoma 14.5+ (Apple Silicon recommended)
- Xcode 16.0+ with visionOS SDK 2.0+
- Swift 6.0+
- Apple Vision Pro or visionOS Simulator

### Installation

```bash
# Clone the repository
git clone https://github.com/akaash-nigam/visionOS_retail-space-optimizer.git
cd visionOS_retail-space-optimizer

# Open in Xcode
open RetailSpaceOptimizer/RetailSpaceOptimizer.xcodeproj

# Build and run
# Select "RetailSpaceOptimizer" scheme
# Choose visionOS Simulator or your Vision Pro device
# Press Cmd+R to build and run
```

### First Launch

1. The app starts with mock data in DEBUG mode
2. Browse the sample stores in the main window
3. Click "Edit Layout" to open the 2D store editor
4. Click "3D Preview" to see the volumetric view
5. Click "Walkthrough" to enter immersive mode

**See [TECHNICAL_README.md](TECHNICAL_README.md) for detailed setup instructions.**

## Executive Summary

Retail Space Optimizer revolutionizes retail store design, merchandising, and customer experience optimization through immersive 3D environments where retailers can visualize, test, and optimize store layouts, product placements, and customer journeys using spatial computing. Built for Apple Vision Pro, this platform transforms traditional retail planning into data-driven, immersive experiences that maximize sales performance and customer satisfaction.

## Problem/Solution

**Problem**: Retailers struggle with optimizing store layouts, product placements, and customer flow patterns using traditional 2D planning tools. Poor spatial design leads to missed sales opportunities, inefficient operations, and suboptimal customer experiences that impact revenue and brand perception.

**Solution**: Retail Space Optimizer creates immersive 3D store environments where retailers can visualize customer journeys, test merchandising strategies, and optimize layouts in real-time. Transform retail planning from guesswork to data-driven optimization that measurably improves business outcomes.
