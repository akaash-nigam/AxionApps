BUILD PATTERNS DOCUMENTED
==========================================

1. **Top-level Package.swift**:
   ```bash
   cd visionOS_AppName
   xcodebuild -scheme SchemeName -sdk xrsimulator \
     -destination 'platform=visionOS Simulator,id=<ID>' \
     -skipPackagePluginValidation -skipMacroValidation build
   ```

2. **Subdirectory Package.swift**:
   ```bash
   cd visionOS_AppName/AppName
   xcodebuild -scheme SchemeName -sdk xrsimulator \
     -destination 'platform=visionOS Simulator,id=<ID>' \
     -skipPackagePluginValidation -skipMacroValidation build
   ```

3. **With .xcodeproj**:
   ```bash
   cd visionOS_AppName
   xcodebuild -sdk xrsimulator \
     -destination 'platform=visionOS Simulator,name=Apple Vision Pro' \
     -skipPackagePluginValidation -skipMacroValidation build
   ```
