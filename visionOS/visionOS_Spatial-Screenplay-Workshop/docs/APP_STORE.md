## App Store Review Notes

### Test Account
Not required - all features available without login

### Notes for Reviewer

Thank you for reviewing Spatial Screenplay Workshop!

**Testing Instructions:**

1. Launch app on Apple Vision Pro simulator or device
2. Tap "New Project" to create sample screenplay
3. View 3D timeline - scene cards should be visible in spatial layout
4. Tap a scene card to select (should highlight)
5. Double-tap to open editor
6. Type some dialogue - auto-formatting should apply
7. Tap "i" button or press Cmd+I for metadata panel
8. Return to timeline, drag a scene to reorder
9. Export PDF from project settings

**Performance Notes:**
- App maintains 60+ FPS with 50 scene cards
- Memory usage < 500MB for typical projects
- Auto-save happens in background every 5 minutes

**Known Limitations:**
- PDF export currently supports up to 200 pages
- Maximum 500 scenes per project (sufficient for feature films)

**Future Updates:**
- iCloud sync (v1.1 - requires CloudKit entitlement)
- Real-time collaboration (v1.1)
- Fountain import (v1.2)
