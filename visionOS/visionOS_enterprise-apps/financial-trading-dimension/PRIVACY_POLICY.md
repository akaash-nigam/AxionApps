# Privacy Policy - Financial Trading Dimension

**Effective Date**: [To be determined]

## Introduction

Financial Trading Dimension ("we", "our", or "the App") is committed to protecting your privacy and ensuring the security of your financial data. This Privacy Policy explains how we collect, use, share, and protect your information when you use our visionOS application.

**By using Financial Trading Dimension, you agree to this Privacy Policy.**

---

## Information We Collect

### 1. Account Information

When you create an account, we collect:
- Name and email address
- Authentication credentials (stored securely, never in plain text)
- User preferences and settings
- Account verification information (if applicable)

### 2. Financial Data

To provide trading and portfolio management services:
- **Portfolio Holdings**: Symbols, quantities, cost basis, current positions
- **Trading Activity**: Order history, executions, cancellations
- **Performance Data**: P&L calculations, returns, risk metrics
- **Watchlists**: Symbols you track and monitor
- **Alerts**: Price alerts and notification preferences

### 3. Market Data Usage

We track:
- Symbols you view and analyze
- Charts and visualizations you interact with
- Technical indicators you use
- Time spent analyzing different assets

### 4. Device and Usage Information

Automatically collected:
- **Device Information**: Vision Pro model, visionOS version
- **App Usage**: Features used, session duration, interaction patterns
- **Performance Metrics**: Frame rates, load times, error occurrences
- **Diagnostic Data**: Crash reports, error logs (anonymized)

### 5. Spatial Data

For immersive experiences:
- **Eye Tracking**: Used locally for interface interaction only
- **Hand Tracking**: Used locally for gesture recognition only
- **Spatial Positioning**: Window and content placement (stored locally)

**IMPORTANT**: Spatial data (eye tracking, hand tracking) is processed entirely on-device and is NEVER transmitted to our servers.

### 6. Collaboration Data (When Using SharePlay)

During collaborative sessions:
- User presence indicators
- Shared annotations and markings
- Voice data (transmitted peer-to-peer via Apple's SharePlay)
- Synchronized view states

## Data Security

### Security Measures

We implement industry-standard security practices:

**Encryption:**
- All data in transit uses TLS 1.3+ encryption
- Sensitive data at rest is encrypted using AES-256
- Passwords are hashed using bcrypt with salt

**Authentication:**
- Biometric authentication (Optic ID) support
- Multi-factor authentication available
- Session timeout for inactive accounts
- Secure token-based authentication

**Access Controls:**
- Role-based access control
- Principle of least privilege
- Regular access audits
- Secure key management

**Infrastructure:**
- Regular security audits and penetration testing
- Continuous security monitoring
- Incident response procedures
- Regular security updates

**Compliance:**
- SOC 2 Type II compliant (if applicable)
- PCI DSS compliant for payment data (if applicable)
- Financial industry security standards

### Financial Data Protection

Given the sensitive nature of financial data:
- Trading credentials are encrypted at rest and in transit
- Order data is protected with additional encryption layers
- Audit logs are maintained for all trading activity
- Regular backup and disaster recovery procedures

## Data Retention

### How Long We Keep Data

| Data Type | Retention Period | Reason |
|-----------|------------------|--------|
| Account Information | Account lifetime + 7 years | Legal requirements |
| Trading History | 7 years after transaction | Regulatory compliance |
| Portfolio Data | Account lifetime | Service provision |
| Usage Analytics | 2 years | Performance improvement |
| Crash Reports | 1 year | Bug fixing |
| Customer Support | 3 years | Support history |
| Audit Logs | 7 years | Compliance |

### Data Deletion

You can request deletion of your data at any time, subject to:
- Legal retention requirements (e.g., 7-year trading history)
- Ongoing legal obligations
- Pending transactions or disputes

After account closure, we will:
1. Delete or anonymize personal information (within 90 days)
2. Retain trading records as required by law (7 years)
3. Remove you from marketing communications (immediately)
