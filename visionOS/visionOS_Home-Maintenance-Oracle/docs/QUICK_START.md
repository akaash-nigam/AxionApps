## Common Questions

**Q: Scanning didn't work?**  
A: Try better lighting, get closer, or use "Add Manually"

**Q: Where's my data stored?**  
A: Locally on your device (+ iCloud if enabled)

**Q: How accurate is recognition?**  
A: 95%+ on common models. Always verify!

**Q: Can I add appliances manually?**  
A: Yes! Tap "Add Manually" instead of scanning
