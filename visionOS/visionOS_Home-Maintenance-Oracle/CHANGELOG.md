### Added

- Automated maintenance schedules

- Service history tracking
- Warranty monitoring
- iCloud sync (optional)
- Dark mode support
- VoiceOver accessibility
- Offline functionality

- **Epic 1: Recognition** - ML scanning, manual entry
- **Epic 3: Inventory** - Appliance management, photos
- **Epic 4: Maintenance** - Scheduling, reminders, history

### Technical

- 152+ tests (Unit, Integration, UI, Performance)
- visionOS 2.0+ support
- Core Data persistence
- On-device ML processing
