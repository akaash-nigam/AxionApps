# Spatial Wellness Platform
*Immersive Employee Health and Wellbeing Ecosystem*

## Executive Summary

Spatial Wellness Platform transforms corporate wellness and employee health programs through immersive 3D environments where employees can engage in mindfulness practices, fitness activities, and wellness education using spatial computing. Built for Apple Vision Pro, this platform creates holistic wellness experiences that improve employee health, reduce stress, and enhance workplace satisfaction through innovative spatial wellness interventions.

## Problem/Solution

**Problem**: Traditional corporate wellness programs suffer from low engagement, limited accessibility, and difficulty measuring impact on employee health and productivity. Employees struggle with work-life balance, stress management, and maintaining healthy habits in demanding work environments, leading to burnout, decreased performance, and increased healthcare costs.

**Solution**: Spatial Wellness Platform creates immersive wellness environments where stress reduction becomes experiential, fitness activities are engaging and accessible, and healthy habits are reinforced through spatial cues and community. Transform corporate wellness from optional programs to integral workplace experiences that demonstrably improve employee health and organizational performance.

## Getting Started

### Prerequisites

- **macOS 14.0+** (Sonoma or later)
- **Xcode 16.0+** with visionOS SDK
- **visionOS 2.0+** simulator or Apple Vision Pro device
- **Swift 6.0+**
- **Node.js 20+** (for landing page development)

### Clone the Repository

```bash
git clone https://github.com/akaash-nigam/visionOS_spatial-wellness-platform.git
cd visionOS_spatial-wellness-platform
```

### Build the visionOS App

```bash
cd SpatialWellness
swift build
```

Or open in Xcode:

```bash
open SpatialWellness/Package.swift
```

Then select **Product > Run** (âŒ˜R) to launch in the visionOS simulator.

### Run Tests

```bash
cd SpatialWellness
swift test
```

For detailed test results, see [TEST_RESULTS.md](TEST_RESULTS.md).

### View Landing Page

```bash
cd landing-page
open index.html
```

Or serve with a local web server:

```bash
cd landing-page
python3 -m http.server 8000
# Visit http://localhost:8000
```
