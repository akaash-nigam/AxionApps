# Changelog

and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-15

### Added
- Initial release of Tactical Team Shooters for Apple Vision Pro
- Core 5v5 competitive gameplay
- Bomb Defusal game mode
- Team Deathmatch game mode
- 4 competitive maps (Warehouse, Office, Construction, Shipyard)
- 6 weapon types (AK-47, M4A1, AWP, Glock, MP9, Nova)
- Hand gesture controls for weapon handling
- Eye tracking for precision aiming
- Spatial audio with directional sound
- Room mapping and dynamic cover generation
- Competitive ranking system (Recruit to Legend)
- ELO-based matchmaking
- Player statistics tracking
- Loadout customization
- Equipment system (grenades, armor, defuse kit)
- Realistic ballistics with bullet drop and travel time
- Client-side prediction with server reconciliation
- Multiplayer networking via MultipeerConnectivity
- Responsive 120 FPS gameplay
- Immersive visionOS interface

### Technical
- Swift 6.0 with strict concurrency
- RealityKit for 3D rendering
- ARKit for spatial tracking
- SwiftUI for UI
- Entity-Component-System architecture
- Performance optimizations for 120 FPS
